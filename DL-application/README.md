# DL-application

## Installation

1. Create a virtual environment:
```bash
python -m venv venv
```

2. Activate the virtual environment:
- On macOS/Linux:
```bash
source venv/bin/activate
```
- On Windows:
```PowerShell
venv\Scripts\activate.ps1
```

3. Install requirements:
```bash
pip install -r requirements.txt
```

## Usage

Run the main application:
```bash
python main.py
